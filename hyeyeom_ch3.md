# 3.1 자바스크립트 실행 환경
브라우저 외부에서 자바스크립트 개발 환경을 제공하는 것이 주 목적인 Node.js는 DOM API 제공하지 않는다. 
브라우저 외부 환경에서는 HTML 요소를 파싱해서 객체화 한 DOM을 직접 다룰 필요가 없기 때문
보안상의 이유로 브라우저 환경의 자 바스크립트는 파일 시스템을 제공하지 않는다.
