# 3장 자바스크립트 개발 환경과 실행 방법

## 3.1 자바스크립트 실행 환경

- 모든 브라우저는 자바스크립트를 해석하고 실행할 수 있는 엔진을 내장함
- Node.js도 엔진을 내장함
- 브라우저는 웹페이지를 브라우저에 렌더링 하는 것이 목적
- Node.js는 브라우저 외부에서 자바스크립트를 실행하는 것이 목적
- 브라우저와 Node.js는 ECMAScript 사양 외의 기능은 기본적으로 지원하지 않음
