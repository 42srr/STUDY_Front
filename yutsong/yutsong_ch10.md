# 10장 객체 리터럴

## 10.1 객체란?

- 자바스크립트는 객체 기반의 프로그래밍 언어
- 자바스크립트는 객체로 이루어져있음
- 단, 원시 값은 객체가 아님

## 10.2 객체 리터럴에 의한 객체 생성

- 보통 객체 리터럴로 객체를 생성함

## 10.3 프로퍼티

- 객체는 프로퍼티의 집합
- 프로퍼티는 키와 값으로 이루어짐
- 키는 값에 접근하기 위한 식별자 역할
- 식별자 네이밍 규칙을 따를 경우 따옴표로 묶지 않아도 됨

## 10.4 메서드

- 자바스크립트의 함수는 일급 객체임
- 함수를 값으로 취급할 수 있음
- 메서드 = 객체의 값으로 묶여있는 함수

## 10.5 프로퍼티 접근

- 프로퍼티에 접근하는 방법은 마침표 표기법, 대괄호 표기법이 있음
- 대괄호 프로퍼티 접근 연산자를 사용할 경우 키값을 따옴표로 감싸줘야 함

## 10.7 프로퍼티 동적 생성

- 존재하지 않는 프로퍼티에 값을 할당하면 동적 생성

## 10.8 프로퍼티 삭제

- delete 연산자로 삭제 가능

## 10.9 ES6에서 추가된 객체 리터럴의 확장 기능

### 10.9.1 프로퍼티 축약 표현

- 프로퍼티 값으로 변수를 사용할 경우, 변수명과 키가 동일하면 키를 생략 가능

### 10.9.2 계산된 프로퍼티 이름

- 표현식으로 키를 동적 생성할수도 있음
- 이때 표현식을 대괄호로 묶어야 프로퍼티 키로 사용할 수 있음

### 10.9.3 메서드 축약 표현

- ES6에서는 메서드 정의 시 function 키워드 생략 가능
