# 18장 함수와 일급 객체

## 18.1 일급 객체
다음과 같은 조건을 만족하는 객체를 '일급 객체'라고 함
- 런타임에 생성 가능(무명 리터럴로 생성가능)
- 변수나 자료구조(객체, 배열 등)에 저장 가능
- 함수 매개변수에 전달가능
- 함수 반환값으로 사용 가능
